# Study02 - 멀티테넌시와 권한 제어 학습

## 1) 왜 멀티테넌시가 중요한가
금융/엔터프라이즈 환경에서는 조직별 데이터 격리가 핵심입니다.

## 2) 이 저장소의 방식
- JWT에 tenant/dept 권한 포함
- 컬렉션을 `bankrag__tenant__dept` 패턴으로 분리
- API Gateway에서 사전 권한 검증

## 3) 실습 아이디어
- `teller` 계정으로 `risk` 부서 데이터 접근 시도 → 차단 확인
- 동일 질문을 tenant별로 실행해 검색 결과 격리 확인

## 4) 심화
- RBAC + ABAC 혼합 모델
- 감사로그와 접근통제 이벤트 연계
