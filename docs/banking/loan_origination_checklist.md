# 여신(대출) 접수/심사 프로세스 요약 (샘플)

## 접수 단계(Front)
- 고객 요청: 목적(주택/사업/운영자금), 희망 한도/기간/상환방식
- 기본 서류: 소득/매출, 재직/사업영위, 신용정보 동의
- 담보/보증: 담보물 확인, 감정/권리관계(근저당, 선순위 등)

## 심사 단계(Credit)
- 상환능력(DSR/DTI 또는 내부 기준)
- 신용등급/연체 이력/부채 현황
- 업종/거래처 집중도(기업)
- 스트레스 테스트(금리 상승, 매출 하락 시나리오)
- 조건부 승인 시 조건 명시(추가 서류, 담보 보강 등)

## 약정/실행(Back)
- 약정서 작성/서명, 약관 교부
- 담보 설정/보증서 발급 확인
- 실행 전 최종 체크: 계좌, 목적 확인, AML 모니터링

## 사후관리
- 한도/만기/금리 변경 이력
- 조기경보(EWS): 연체 조짐, 매출 급감, 담보 가치 하락
