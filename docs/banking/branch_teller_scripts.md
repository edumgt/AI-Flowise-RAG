# 지점 창구 응대 스크립트 예시 (샘플)

## 본인확인 요청
- “안전한 거래를 위해 신분증 확인을 부탁드립니다.”
- “성함과 생년월일 확인 부탁드립니다.”

## 수수료/한도 안내
- “해당 거래는 1일 한도가 있으며, 초과 시 추가 확인 절차가 필요합니다.”
- “수수료는 고객 등급/상품에 따라 달라질 수 있습니다.”

## 민원 대응(기본)
- “불편을 드려 죄송합니다. 정확한 확인 후 안내드리겠습니다.”
- “처리 경과를 공유드리기 위해 연락 가능한 번호를 확인해도 될까요?”

## 보이스피싱 의심 시
- “최근 금융사기 피해가 많아 몇 가지 확인 질문을 드립니다.”
- “송금 목적과 수취인과의 관계를 확인할 수 있을까요?”
- “급하게 진행하라는 요청이 있었는지 확인 부탁드립니다.”
